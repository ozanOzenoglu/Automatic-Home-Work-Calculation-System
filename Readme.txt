This bash-scriptted program is developed for calculating students homework automatically. This program firstly downloads a zipped homework file from the online submit-system.  Then it unzips   the zipped file to  check the expected hierarchy of folders and all files to compare the contents with the desired content..

  After the checking process , the system calculates the remark of the homework and sends the final point to the students email addresses  . This system is written with Linux bash scripts and can be used as a real automatic calculator system if needed.

This is a free system to use only if you share the all files with public including this readme.txt file. 

Bash script kullanilarak kodlanmis bu sistem ögrencilerin submit sisteme yükledikleri ödev dosyalarini otomatik olarak puanlarini hesaplar. Sistemden ziplenmis dosyayi çeken sistem daha sonra dosyayi unzip leyerek içindeki yapiyi beklenen yapi ve dosyalarin içeriginide beklenen içerikle karsilastirdiktan sonra
hesaplanan puani ögrenciye mail olarak atar.


Bu sistemi ücretsiz olarak kullanabilmeniz için bütün dosyalari bu readme.txt dosyasi dahil olmak üzere açik alanda paylasamniz gerekmektedir.
